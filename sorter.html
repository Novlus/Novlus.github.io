This year during my work-study program I had to create a file sorter for the Human Resources Management team of the Territorial Department of Digital Transformation and Biomedical Engineering (DTNIB) of the CHRU of Nancy
it's a standby application that I had to make work again. I had to find the version of java on which this application was running and then find the reason why the program no longer worked. I then had to make modifications so that the program was compatible with a new database.

I will still explain to you how the program works. This program works with several parameters. You will need a txt file containing the list of all the files to be sorted.
You must also give as a parameter the directory you want to sort and a directory to contain all the files to sort.
the sorted file are pdf. These pdfs contain in their name a type, an employee number, a surname and a first name.
Thanks to this information we can sort the different files. If this information is incorrect compared to what is in the database, the file will not be sorted and its extension will be renamed with an error message.
