Mediatek Project for the IUT Nancy Charlemagne

The mediatek project was realized in Javascript. This project allowed me to improve in javascript especially in DOM manipulation. The goal of this project is to make a mediatek for movies, series and video games. You can sort with a filter by media type, you can add media, modify them and delete them.
