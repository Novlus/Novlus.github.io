Unlock Project for the IUT Nancy Charlemagne

The unlock project was realized in PHP a server side language. This project allowed us to discover a PHP framework called Slim. This framework allows to manage the routes in PHP. The goal of the unlock project is to convert the interactions with the maps in our php project. The unlock project allows to play the game with the unlock application available on the Playstore.

Now let's move on to the explanation of the unlock program. You will notice that there are several fields. The first one allows you to draw a card if you know its number.
The second field allows you to discard a card, be careful not to discard a card that could still be useful, otherwise you will have to start your game over. Finally, the last two fields allow you to combine two cards, if both are combinable. If not, an error message will appear. If the cards can be combined then a new card, a number, a bonus or a penalty will appear.
